# Peec Channel Report — 2026-04-16

One-page channel attribution snapshot for peec.ai, built from GA4, Google Search Console, PostHog, demo-booking self-report, LinkedIn analytics, and Radyant AI-search data.

Live report: https://noahw13.github.io/peec-channel-report/

## What is in it

- GA4 session and key-event channel mix (Apr 2025 to Apr 2026)
- Demo self-report vs GA4 side-by-side
- Working decomposition of the Direct bucket
- AI-search monthly trend and LLM source split
- Ranked channel table with confidence tags
- Caveats and next steps

## Files

- `index.html` — standalone report, Chart.js via CDN

## Caveats

Numbers are directional. Cross-subdomain distinct-ID merge on PostHog is incomplete, LLM referrers are undercounted by GA4, and UTM capture on signups is ~5%. See the caveats section in the report.
